## 📱 Configuración por Dispositivo

### 📱 **iPhone/iPad (iOS)** {#iphone-express}

#### **Configuración Express (5 minutos):**
1. **Configuración** → **Tiempo en Pantalla**
2. **Activar Tiempo en Pantalla** → **Es el iPhone de mi hijo**
3. **Crear código de familia** (diferente al código del dispositivo)
4. **Límites de App** → Agregar límite → **Todas las Apps y Categorías** → 1-2 horas

#### **Configuración Completa:**
- **[📱 Guía Completa iOS](/configuracion-ios)** - Configuración avanzada paso a paso
- **[👶 iOS para Niños Pequeños](/ios-ninos-pequenos)** - Configuración 0-6 años
- **[🚀 iOS para Adolescentes](/ios-adolescentes)** - Configuración 11-18 años

### 🤖 **Android** {#android-express}

#### **Configuración Express (5 minutos):**
1. **Instalar Family Link** de Google Play Store
2. **Crear cuenta supervisada** para tu hijo
3. **Configurar límites diarios** → 1-2 horas
4. **Activar filtros** → Contenido restringido

#### **Configuración Completa:**
- **[🤖 Guía Completa Android](/configuracion-android)** - Family Link avanzado
- **[📱 Samsung Kids](/samsung-kids)** - Configuración específica Samsung
- **[🔒 Android Empresarial](/android-empresarial)** - Configuración MDM

## 🌐 Control de Red WiFi {#router-express}

<div class="emergency-section">

### 🚨 **Configuración de Emergencia (5 minutos)**

Si necesitas **bloquear contenido AHORA** en toda tu red:

1. **Cambiar DNS** en tu router:
   - DNS Primario: `1.1.1.3` (Cloudflare Family)
   - DNS Secundario: `1.0.0.3`

2. **O usar DNS de OpenDNS:**
   - DNS Primario: `208.67.222.123`
   - DNS Secundario: `208.67.220.123`

**[🌐 Guía Completa Control de Red](/control-red-hogares)**

</div>

### **Control Avanzado de Red:**
- **Filtrado por categorías** (redes sociales, gaming, adulto)
- **Horarios de internet** (sin internet después de 9pm)
- **Dispositivos específicos** (solo tablet del niño)
- **Monitoreo de actividad** (qué sitios visita)

## ✅ Checklist de Implementación

<div class="code-example">

### **📋 Lista de Verificación Básica:**

**Configuración Inicial (30 minutos):**
- [ ] Configurar controles en dispositivo principal del niño
- [ ] Establecer límites de tiempo básicos
- [ ] Activar filtros de contenido apropiados para la edad
- [ ] Configurar DNS seguro en router WiFi

**Conversaciones Familiares (15 minutos):**
- [ ] Explicar al niño las nuevas reglas digitales
- [ ] Establecer consecuencias claras
- [ ] Acordar horarios sin dispositivos (comidas, antes de dormir)
- [ ] Crear plan de actividades alternativas

**Verificación Semanal (10 minutos):**
- [ ] Revisar reportes de tiempo de pantalla
- [ ] Verificar que controles siguen activos
- [ ] Ajustar límites según comportamiento
- [ ] Conversar sobre experiencia digital de la semana

</div>
